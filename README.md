# dart-cli-webscrape-google

## Description
Webscrapes google's homepage.
Returns html source.

## Tech stack
- dart
  - http

## Docker stack
- google/dart

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
